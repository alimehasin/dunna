# API overview

Everything hangs off the **`dunna`** object returned by `import { dunna } from 'dunna'`.

| Namespace                             | Purpose                                   |
| ------------------------------------- | ----------------------------------------- |
| [`dunna.basic`](/modules/basic)       | Booleans, numbers, letters, array picks   |
| [`dunna.color`](/modules/color)       | Named colors, hex, RGB objects            |
| [`dunna.datetime`](/modules/datetime) | Past, future, ranges, birthdates          |
| [`dunna.finance`](/modules/finance)   | Currency codes and names                  |
| [`dunna.location`](/modules/location) | Countries, cities, coordinates            |
| [`dunna.person`](/modules/person)     | Names, gender, email, full person objects |

Open a module page for signatures, options tables, and examples.
