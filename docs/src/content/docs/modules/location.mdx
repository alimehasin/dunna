This is the location module, it contains countryName, countryCode, city, longitude, latitude

## CountryName

`dunna.location.countryName()` returns a random country name

```js
dunna.countryName();
```

## CountryCode

`dunna.location.countryCode()` returns a random country code

```js
dunna.countryCode();
```

## City

`dunna.location.city()` returns a random city

```js
dunna.city();
```

## Longitude

`dunna.location.lng()` returns a random longitude

```js
dunna.lng();
```

## Latitude

`dunna.location.lat()` returns a random latitude

```js
dunna.lat();
```
