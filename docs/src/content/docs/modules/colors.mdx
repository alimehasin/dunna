This is the color module, it colorName, colorHexCode, colorRGBCode, color

## ColorName

`dunna.color.name()` returns a random color name

```js
dunna.color.name();
```

## ColorHexCode

`dunna.color.hex()` returns a random color code

```js
dunna.color.hex();
```

## ColorRGBCode

`dunna.color.rgb()` returns a random color code

```js
dunna.color.rgb();
```

## Color

`dunna.color.color()` returns a random color

```js
dunna.color.color();
```
