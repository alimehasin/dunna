This is the colors module, it colorName, colorHexCode, colorRGBCode, color

## ColorName

`dunna.colors.name()` returns a random color name

```js
dunna.colors.name();
```

## ColorHexCode

`dunna.colors.hex()` returns a random color code

```js
dunna.colors.hex();
```

## ColorRGBCode

`dunna.colors.rgb()` returns a random color code

```js
dunna.colors.rgb();
```

## Color

`dunna.colors.color()` returns a random color

```js
dunna.colors.color();
```
