This is the finance module, it currencyName, currencyCode, currency

## CurrencyName

`dunna.finance.currencyName()` returns a random currency name

```ts
dunna.currencyName();
```

## CurrencyCode

`dunna.finance.currencyCode()` returns a random currency code

```ts
dunna.currencyCode();
```

## Currency

`dunna.finance.currency()` returns a random currency

```ts
dunna.currency();
```
