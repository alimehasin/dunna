# Basic

## Boolean

`dunna.boolean()` returns a random boolean (`true` or `false`)

```js
// Use default likelihood = 50
dunna.boolean(); // true
dunna.boolean(); // false
dunna.boolean(); // false

// Modify likelihood
dunna.boolean({ likelihood: 80 }); // The output has chance of 80% to be true
dunna.boolean({ likelihood: 20 }); // The output has chance of 20% to be true
```

<Generator
  data={[
    {
      name: 'likelihood',
      description: 'The likelihood of true occurrence',
      type: 'number',
      initialValue: 50,
    },
  ]}
  onClick={(dunna, state, theme) => {
    const bool = dunna.boolean({ likelihood: state.likelihood });
    const text = bool.toString();
    const styles = {
      backgroundColor: bool ? theme.colors.green[6] : theme.colors.red[6],
    };
    return { text, styles };
  }}
/>

## Integer

`dunna.integer()` returns a random integer between 0 (inclusive) and 10 (bot inclusive)

```js
dunna.integer(); // 4
dunna.integer(); // 7
dunna.integer(); // 4

// Customize parameters
dunna.integer({ min: 1, max: 100 });
dunna.integer({ min: -10, max: 10 });
```

<Generator
  data={[
    {
      name: 'min',
      description: 'The minimum number `inclusive`',
      type: 'number',
      initialValue: 0,
    },
    {
      name: 'max',
      description: 'The maximum number `not inclusive`',
      type: 'number',
      initialValue: 10,
    },
  ]}
  onClick={(dunna, state, theme) => {
    const integer = dunna.integer({ min: state.min, max: state.max });
    const text = integer.toString();
    return { text };
  }}
/>

## Float

`dunna.float()` returns a random float number between 0 (inclusive) and 10 (not inclusive)

```js
dunna.float(); // 4.364
dunna.float(); // 8.834

// Parameters
dunna.float({ min: 3, max: 36, fixed: 5 });
```

<Generator
  data={[
    {
      name: 'min',

      type: 'number',
      initialValue: 0,
    },
    {
      name: 'max',
      description: 'The maximum number `not inclusive`',
      type: 'number',
      initialValue: 10,
    },
    {
      name: 'fixed',
      description: 'count of digits after the floating point`',

  }}
/>

## Letter

`dunna.letter()` returns a random uppercase or lowercase letter

```js
dunna.letter(); // h
dunna.letter(); // E

// Parameters
dunna.letter({ casing: 'upper' });
```

<Generator
  data={[
    {
      name: 'casing',
      description: 'Specify the casing of the letter',
      type: 'enum',
      initialValue: 'any',
      values: ['any', 'upper', 'lower'],
    },
  ]}
  onClick={(dunna, state, theme) => {
    const text = dunna.letter({ casing: state.casing });
    const styles = {
      backgroundColor:
        text === text.toUpperCase()
          ? theme.colors.blue[8]
          : theme.colors.blue[4],
    };
    return { text, styles };
  }}
/>

## Choice

`dunna.choice(array)` returns a random element from an array

```js
dunna.choice(['Alice', 'Bob', 'Charlie']); // Alice
dunna.choice(['Alice', 'Bob', 'Charlie']); // Charlie
```

## Pick

`dunna.pick(count, array)` returns a random subset of the given array with the given count

```js
dunna.pick(3, [2, 1, 6, 3, 4]); // [6, 4, 1]
dunna.pick(2, ['Alice', 'Bob', 'Charlie']); // ['Bob', 'Charlie']
```

## Omit

`dunna.omit(count, array)` returns a random subset of the given array by omitting elements based on the given count

```js
dunna.omit(2, [2, 1, 6, 3, 4]); // [1, 6 ,4]
dunna.omit(2, ['Alice', 'Bob', 'Charlie']); // ['Bob']
```
